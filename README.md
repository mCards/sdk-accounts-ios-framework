# sdk-accounts-ios-framework
SDK containing common model classes used to show accounts

## Importing
The SDK can be imported via SPM (Swift Package Manager).
- Add the package in the dependencies section of the Swift Manifest file. For example, `.package(url: "https://github.com/mCards/sdk-accounts-ios-framework", from: "1.0.0")`
- In the tartet dependencies section add: `.product(name: "AccountsSDK", package: "sdk-accounts-ios-framework")`
